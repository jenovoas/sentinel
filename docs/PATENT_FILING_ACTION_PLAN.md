**Prior Art:** US12130917B1 (generic prompt injection)

**Our Differentiation:**
```
PRIOR ART SCOPE:
"GenAI prompt injection classifier for user text inputs"

OUR SCOPE:
"Multi-modal telemetry sanitization for autonomous IT operations"

KEY DIFFERENCES:
1. Input type: Telemetry (logs+metrics+traces) vs user text
2. Context: AIOps automation vs GenAI chatbots
3. Sources: Prometheus, Loki, Tempo, Auditd vs single text input
4. Validation: Multi-source correlation vs single classifier
5. Domain: IT operations vs conversational AI

CLAIM LANGUAGE (Narrow Scope):
"A method for securing autonomous IT operations against adversarial 
telemetry injection, comprising receiving operational telemetry data 
from a convergent observability stack comprising at least time-series 
metrics, structured logs, distributed traces, and kernel-level audit 
events..."
```

**Prior Art:** US12248883B1 (generic malicious prompt detection)

**Our Differentiation:**
```
PRIOR ART SCOPE:
"Generative AI model prompt injection classifier"

"Multi-factor decision engine with Bayesian confidence scoring"

1. Architecture: Dual guardians vs single classifier

3. Scoring: Bayesian inference vs simple risk score
4. Context: Admin ops, DR mode, maintenance windows
5. Validation: Both guardians must confirm

"A multi-factor decision engine for autonomous security remediation, 
comprising a telemetry aggregator configured to collect events from 
heterogeneous sources including kernel-level syscall monitors, 
application-level log aggregators, infrastructure metrics collectors,
```
